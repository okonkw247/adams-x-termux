# 👑 ADAM'S X - Termux Setup

> A professional developer terminal setup for Android using Termux

## ✨ Features
- 🎨 Dracula color theme
- ⚡ Powerlevel10k ZSH prompt
- 🌊 Matrix rain startup animation
- 📁 LSD file manager with icons
- 🤖 AI assistant powered by Groq
- 💻 Neovim with AI plugin
- 🚀 Tmux with custom status bar
- 🔧 Git configured and ready

## 📦 What gets installed
- ZSH + Oh My Zsh + Powerlevel10k
- Neovim, Tmux, Git
- Node.js, Python, Rust
- LSD, Fastfetch, Figlet, Toilet

## 🚀 Install
\`\`\`bash
curl -fsSL https://raw.githubusercontent.com/okonkw247/adams-x-termux/main/setup.sh | bash
\`\`\`

## 📱 Requirements
- Android 7+
- Termux from F-Droid (not Play Store)
- Internet connection
